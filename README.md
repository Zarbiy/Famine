# 🔏 Famine - 42 Crypto Project

Projet de l’école **42** : créer un binaire qui **modifie d’autres binaires** pour y ajouter une signature unique, sans altérer leur comportement.

---

## 📖 Description

L’objectif du projet **Famine** est :  
- Concevoir un binaire `famine` capable de modifier **un ou plusieurs binaires**  
- Ajouter une **signature personnalisée**
- S’assurer que le **comportement original du binaire reste inchangé**  
- Appliquer cette signature à **tous les binaires présents dans un dossier temporaire**
